Create professional-looking documents by combining daisy wheel printing technology with easy to use word processing software. Type with confidence. The 7 line display and 3.5" 1.44MB disk drive make word processing easier than ever before.

OVERVIEW

Brother's line of daisywheel word processors include LW-100, LW-200 and LW-350. The LW-100 include 4 line LCD display, 240KB disk drive, LW-200 include 7 line LCD display, 1.44MB disk drive and LW-350 include 14 line LCD display, 1.44MB disk drive.

Powerful yet portable, the design of the LW-200 makes it ideal for the writer on the move. A wide range of editing features provides real word processing power with the familiarity of a typewriter.

SPECIFICATIONS

  • Up to 13 cps Print Speed
  • 326.9mm (12.87") Paper Capacity
  • 228.6mm (9") Typing Capacity
  • Typing Pitches : 10, 12, 15
  • 1, 1.5, 2 Line Spacing
  • Original + 4 Copy Capacity
  • Keyboard with 96 Character Capability
  • 32 KB Text Memory
  • 1 Line Correction Memory
  • 3.5", 1.44MB Internal Disk Drive
  • 80 Characters by 7 Line LCD
  • Spell Checker
  • 204 word User Dictionary
  • Paper Supporter
  • Carrying Handle
  • Unit Dimensions : 408mm (16.1") x 136mm (5.4") x 399mm (15.7")
  • Unit Weight : 7kg
