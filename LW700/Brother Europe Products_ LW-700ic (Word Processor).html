Create professional-looking letters, invitations, reports and much more with the LW-700ic Inkjet Word Processor with colour printing


OVERVIEW

Brother's line of Inkjet Word Processors include both portable and desktop models. This portable model, the LW-700ic, includes a 14 line LCD display, 1.44MB disk drive, Professional Touch Keyboard and built-in colour inkjet printer. The other models in this series include the LW-700i/710i, LW-730i, and LW-750ic.

The LW-700ic allows you to create stylized reports, presentations, newsletters and brochures by combining multiple type styles, fonts, and sizes into a single document. The LW-700ic and LW-750ic incorporate printers which allow you to print in up to 7 different colours. The LW-700i/710i and LW-730i offer monochrome printing only.

OTHER APPLICATIONS

  • Addressbook with Data Merge and label printing for merging names and addresses into envelopes and labels.
  • 100 black graphic art images and 20 color graphic art images including headlines, pictures and holiday greetings.
  • Typewriter mode for typing simple labels, envelopes, etc.
  • Tutorial
  • Help Screens
  • Calculator

SPECIFICATIONS
  • 12" carriage width
  • 9.1" printing width
  • 300 x 300 dpi inkjet printing
  • Up to 160 cps printing speed (Monochrome printing)
  • Unit dimensions: 409mm(16.1") x 147mm(5.8") x 455mm(17.9")
  • Unit weight: 5.9 kg
  • Carton dimensions: 564mm(22.2") x 248mm(9.8") x 550mm(21.7")
  • Carton weight: 8.6 kg

ACCESSORIES
  • IN-700 high capacity black ink cartridge
  • IN-710C color ink cartridge
  • CF-710C cut sheet feeder for automatic paper handling
